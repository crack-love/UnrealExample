# Icon
디스크립터 파일과 함께 플러그인은 에디터 플러그인 브라우저에 표시할 아이콘이 필요합니다. 
이미지는 128x128 .png 파일로 이름은 "Icon128.png" 여야 하며 플러그인의 "/Resources/" 디렉터리에 있어야 합니다.
